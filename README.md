# README

## 그룹 권한 관리 모듈

### 기능 설명

#### 1. 리소스 관리
- #### 리소스 프로그램 관리
- #### 리소스 메뉴 관리

#### 2. 롤 그룹 관리
- #### 롤 그룹을 만들어서 그룹 권한 N개를 부여할 수 있다.

#### 3. 그룹 권한 관리
- #### 그룹 권한을 만들어서 리소스 권한 N개를 부여할 수 있다.

#### 4. 리소스 권한 관리
- #### 등록된 리소스에 대해서 권한 관리를 할 수 있다.
- #### 권한은 접근권한, 수정권한으로 한정한다.

#### 5. 유저, 회사 관리
- #### 유저와 회사를 등록 할 수 있고, 관리할 수 있다.
- #### 유저에 롤 그룹을 부여해서 권한을 부여할 수 있다.
